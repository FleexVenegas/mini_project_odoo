# Weighted Cost Module

## Descripción

Módulo de configuración para Costo Ponderado en Odoo 17.

## Características

- Configuración accesible desde **Ajustes → Costo Ponderado**
- Hereda de `res.config.settings`
- Opciones de configuración:
  - **Habilitar Costo Ponderado**: Activa el sistema de costo ponderado
  - **Cálculo Automático**: Calcula automáticamente con los movimientos de stock
  - **Precisión Decimal**: Define el número de decimales para los cálculos

## Instalación

1. Copiar el módulo en la carpeta `addons`
2. Actualizar la lista de aplicaciones en Odoo
3. Instalar el módulo "Costo Ponderado"

## Uso

1. Ir a **Ajustes → Costo Ponderado**
2. Configurar las opciones según necesidades
3. Guardar cambios

## Dependencias

- base
- stock

## Versión

- Odoo 17.0
- Versión del módulo: 1.0.0
